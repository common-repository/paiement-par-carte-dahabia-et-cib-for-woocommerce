Acceptez les cartes CIB & DAHABIA sur WooCommerce. Extension officielle certifiée par la Satim pour le e-paiement en Algérie.

== Description ==
= 👉 Acceptez les cartes CIB & DAHABIA  =
Cette extension vous permet d’accepter les paiements par Carte **DAHABIA & CIB**. C’est une passerelle de paiement entre l’API de la SATIM Algérie et WooCommerce.

= 👉 Activation API SATIM: 45 000 DA =
L’activation de l’API SATIM nécessite de passer les tests de non-regression selon la réglementation et le cahier des charges de la CIB WEB Algérie. 
Pour y procéder, il vous suffit de:
* [Remplir le formulaire d’activation disponible ici](https://web-rocket.dz/e-paiement-algerie/)
* Payer les frais d’activation de 45 000 DA

= 👉 Avantages =
* Activation express en moins de 48h.
* Extension officielle et certifié par la SATIM / CIB WEB.
* Marche avec les cartes CIB et DAHABIA.
* Compatible avec tous les domaines (examples: .dz, .com, .net, .org)
* Protection contre les risques de fraude, d’abus et d’annulation.
* Possibilité d’activé le “Mode Test”.
* Le montant de la transaction est directement crédité sur votre compte bancaire.

== Installation ==
1. Installer l’extension

2. Activer l’extension.

3. Se rendre à **Woocommerce => Réglages =>  Paiements**

4. Activer la méthode **\"Carte CIB & Dahabia / Satim**

5. Cliquer sur **Manager** 

6. Suivez les **indications en rouge** sur l’écran.
